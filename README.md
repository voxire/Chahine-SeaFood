# Chahine Sea Food

Website for Chahine Sea Food, a Lebanese seafood restaurant. Built with Next.js, Tailwind CSS, and next-intl for Arabic/English bilingual support.

Live: [chahine-sea-food.vercel.app](https://chahine-sea-food.vercel.app)

## Stack

- Next.js 14 (App Router)
- Tailwind CSS
- next-intl (Arabic / English)
- Lenis smooth scroll

## Dev

```bash
pnpm install
cp .env.local.example .env.local
pnpm dev
```
